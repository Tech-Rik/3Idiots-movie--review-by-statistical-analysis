# 3 Idiots Audience Review Analysis

This repository contains a professional Jupyter Notebook for analyzing audience reviews of the Bollywood movie **3 Idiots**.

## Project Structure

- `notebooks/3_idiots_audience_review_analysis.ipynb` — main notebook with statistical analysis, visualizations, and interpretation
- `data/3_idiots_reviews.csv` — review dataset used in the analysis
- `README.md` — project overview and setup instructions

## What the notebook includes

- data loading and overview
- data cleaning and preprocessing
- descriptive statistics
- mean, median, mode, range, variance, standard deviation, and skewness
- frequency distribution
- visualizations such as histogram, bar chart, pie chart, box plot, density plot, and word cloud
- final interpretation and conclusion

## Requirements

Install the following Python packages if needed:

```bash
pip install pandas numpy matplotlib scipy wordcloud ipython
```

## Run the notebook

1. Open the notebook from the `notebooks` folder in Jupyter Notebook or JupyterLab.
2. Run all cells in order.
3. The analysis and plots will be generated automatically.

## Project purpose

This notebook demonstrates how statistical analysis can be used to understand audience opinions, satisfaction levels, and emotional patterns in movie reviews.
